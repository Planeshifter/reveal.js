Semantic Web Technologies

RDF (Resource Description Framework)
  • Allows to define triples of 'subject', 'predicate', 'value'
  • Designed to be read and understood by computers.
  • URIs are used as names for entities.
  • HTTP URIs can be looked up to obtain useful information
  • Allows to deal with name disambiguation problem

OWL (Web Ontology Language)
  • Based on RDF, OWL allows logical statements and inference

Problem Statement:

Inferring the sentiment (approval / disapproval) of congressional speeches towards legislation based on the documents and the implied network structure.

Literature Review:

  • Thomas, M., Pang, B., Lee, L., 2006. Get out the vote: Determining support or opposition from Congressional floor-debate transcripts. Proceedings of EMNLP.
  • Burfoot, C., Bird, S., Baldwin, T., 2011. Collective Classification of Congressional Floor-Debate Transcripts. Comput. Linguist. 1506–1515.
They do Sentiment classification of U.S. Congressional floor-debate transcripts. Their data set is available for further research.
Note that from an experimental point of view, this is a very convenient problem to work with because we can automatically determine ground truth (and thus avoid the need for manual annotation) simply by consulting publicly available voting records.

Collective Classification

  • Ordinary classification methods such as Logistic Regression or Naive Bayes predict the value for actor $Y_i$ using a feature vector $x_i$ containing the observed attributes of the observation
  • In collective classification, one uses also information about actors in the neighbourhood of $Y_i$ in the prediction task
  • It is common for one speaker to make reference to another in agreement.
